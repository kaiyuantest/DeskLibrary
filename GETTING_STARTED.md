# 快速开始

## 安装

### 方式 1：下载安装包（推荐）

1. 访问 [Releases](https://github.com/yourusername/DeskLibrary/releases) 页面
2. 下载最新版本的 `.exe` 安装包
3. 双击运行安装程序
4. 按照提示完成安装

### 方式 2：从源码构建

#### 环境要求
- Windows 10/11
- Node.js 16+
- Git

#### 步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/yourusername/DeskLibrary.git
   cd DeskLibrary
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **运行开发版**
   ```bash
   npm run dev
   ```

4. **构建安装包**
   ```bash
   npm run dist
   ```
   
   构建完成后，安装包位于 `release/` 目录。

## 首次使用

### 1. 启动应用

- 双击桌面图标或从开始菜单启动
- 应用会最小化到系统托盘
- 双击托盘图标打开主窗口

### 2. 基本设置

打开"设置"页面，配置：

- **开机启动**：开启后应用会随系统启动
- **悬浮图标**：显示桌面悬浮球
- **靠边停靠**：主窗口靠边自动隐藏
- **复制后智能判断**：自动判断是否收藏剪贴板内容

### 3. 开始使用

#### 收藏剪贴板内容

1. **自动收藏**
   - 开启"复制后智能判断"
   - 复制任何文本或图片
   - 应用会自动判断是否收藏

2. **手动收藏**
   - 复制内容后，在观察窗口内连按两次复制
   - 或复制后按 Shift/Ctrl/Alt 键

3. **累计复制**
   - 按 `Ctrl+Alt+A` 开始累计
   - 连续复制多段内容
   - 按 `Ctrl+Alt+S` 结束累计
   - 所有内容会合并为一条记录

#### 管理资源

1. **备份存储**
   - 拖入文件到"📦 备份存储"区域
   - 文件会被复制到备份目录
   - 原文件删除后仍可访问

2. **快捷入口**
   - 拖入文件到"🔗 快捷存储"区域
   - 只保存文件路径
   - 快速访问常用文件

#### 导入浏览器 Cookie

1. 点击"离线导入"按钮
2. 选择浏览器类型（Chrome/自建/比特）
3. 扫描并选择要导入的配置文件
4. 点击"确认导入"

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+Z` | 删除上次收藏 |
| `Ctrl+Alt+A` | 开始累计复制 |
| `Ctrl+Alt+S` | 结束累计复制 |

所有快捷键都可以在设置中自定义。

## 常见问题

### Q: 剪贴板监听不工作？

A: 
1. 确保应用正在运行（检查托盘图标）
2. 检查是否开启了相关功能
3. 尝试重启应用

### Q: 如何备份我的数据？

A: 数据保存在：
```
C:\Users\<用户名>\AppData\Roaming\desklibrary-electron\data\
```
复制整个 `data` 文件夹即可备份。

### Q: 如何恢复数据？

A: 
1. 关闭应用
2. 将备份的 `data` 文件夹复制到上述路径
3. 重新启动应用

### Q: 悬浮窗不显示？

A: 在设置中开启"悬浮图标"选项。

### Q: 如何卸载？

A: 
1. 在控制面板中卸载应用
2. 手动删除数据目录（如果需要）

## 进阶使用

### 自定义备份路径

1. 打开设置页面
2. 找到"资源备份存储路径"
3. 点击"选择"按钮选择目录
4. 保存设置

### 批量管理浏览器卡片

1. 在浏览器卡片页面勾选多个卡片
2. 使用批量操作按钮：
   - 批量测试：检查 Cookie 是否有效
   - 批量删除：删除选中的卡片

### 使用悬浮菜单

1. 点击悬浮球打开菜单
2. 快速访问常用功能：
   - 打开主窗口
   - 删除上次收藏
   - 开始/结束累计
